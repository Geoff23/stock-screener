How it began

Back in 2019, I was reading a book called “The Intelligent Investor” by Benjamin Graham. By then, I was already on chapter 14, which introduced seven criteria for selecting stocks.

Naturally, I wondered: what stocks meet all the criteria?

So, I started with a few stocks I knew: Apple, Nvidia, Tesla, etc. I would go on websites like Yahoo Finance and Macrotrends to find statistics for each stock, like market cap and current ratio. Then, I would see if they meet the criteria:

  • Is the market cap at least $2 billion?
  • Is the current ratio at least 2?
  • And so on...

One by one, they got eliminated, and I started running out of stocks I could research. So, I pulled up a list of all the stocks in the S&P 500 Index. Some failed right away, others met all the requirements except one. But after spending hours combing through balance sheets and income statements, I still couldn’t find a single stock that met all the criteria.

Then I thought: I could use stock screeners! That would save me tons of time. Unfortunately, none of them had the criteria listed in the book. The ones that did were obscure and difficult to use. So, I decided to program my own stock screener.

How it works

This application was built using Python and Flask. The website itself uses HTML, CSS, and JavaScript. Each of them serves a different function:

  • HTML sets the page content and structure
  • CSS styles the website and makes it visually appealing
  • Javascript updates the HTML and CSS to make the website interactive

To evaluate stocks, the stock screener first needs data. To do this, the python program extracts the ticker symbols of over 3000 stocks that make up the CRSP US Total Stock Market Index from a CSV file. Then, it uses Beautiful Soup to extract financial data from websites like MacroTrends and Yahoo Finance, before storing the data in an SQLite database.

When a user submits a form through the website, Flask sends the form’s data back to the server, which checks the database to find stocks that meet the given criteria. It then returns all those stocks along with their statistics back to the website, where it is displayed as a table. This is done using a jQuery plug-in called DataTables, which allows the user to sort and compare the stocks.
